## 1-1：最大子数组和

给定一个整数数组 `nums`，请找到和最大的连续子数组，并返回其和。

### 示例
- 输入：`nums = [-2,1,-3,4,-1,2,1,-5,4]` → 输出：`6` → 连续子数组 `[4,-1,2,1]` 的和最大，为 6。
- 输入：`nums = [1]` → 输出：`1` → 连续子数组 `[1]` 的和为 1。
- 输入：`nums = [5,4,-1,7,8]` → 输出：`23` → 连续子数组 `[5,4,-1,7,8]` 的和为 23。

### 约束
- $1 \le \text{nums.length} \le 10^5$
- $-10^4 \le \text{nums}[i] \le 10^4$

### 进阶
在掌握 $O(n)$ 解法（Kadane 算法）后，尝试用分治思想再实现一次。

## 1-2：寻找峰值

峰值元素严格大于其相邻元素。给定一个下标从 0 开始的整数数组 `nums`，找到任意一个峰值元素并返回其下标。可将 $\text{nums}[-1] = \text{nums}[n] = -\infty$。

要求算法时间复杂度为 $O(\log n)$。

### 示例
- 输入：`nums = [1,2,3,1]` → 输出：`2` → `nums[2] = 3` 为峰值。
- 输入：`nums = [1,2,1,3,5,6,4]` → 输出：`5` → `nums[5] = 6` 为峰值（下标 `1` 也可）。

### 约束
- $1 \le \text{nums.length} \le 1000$
- $-2^{31} \le \text{nums}[i] \le 2^{31}-1$
- 对所有有效 $i$，均有 $\text{nums}[i] \ne \text{nums}[i+1]$。

## 1-3：多数元素

给定大小为 `n` 的数组 `nums`，返回出现次数超过 `⌊n/2⌋` 的元素。可假设多数元素必定存在。

### 示例
- 输入：`nums = [3,2,3]` → 输出：`3`
- 输入：`nums = [2,2,1,1,1,2,2]` → 输出：`2`

### 约束
- $n = \text{nums.length}$
- $1 \le n \le 5\times 10^4$
- $-10^9 \le \text{nums}[i] \le 10^9$

### 进阶
能否在线性时间、$O(1)$ 额外空间内完成？（Boyer–Moore 投票法）

## 1-4：寻找重复数

给定一个长度为 `n + 1` 的整数数组 `nums`，其中每个整数都在区间 `[1, n]` 内，数组中仅有一个整数重复出现至少两次。请在不修改数组、且仅使用常数级额外空间的前提下，返回这个重复数。

### 示例
- 输入：`nums = [1,3,4,2,2]` → 输出：`2`
- 输入：`nums = [3,1,3,4,2]` → 输出：`3`
- 输入：`nums = [3,3,3,3,3]` → 输出：`3`

### 约束
- $1 \le n \le 10^5$
- $\text{nums.length} = n + 1$
- $1 \le \text{nums}[i] \le n$
- 除恰有一个整数出现至少两次外，其余整数均只出现一次。

### 进阶
- 为什么一定存在重复数？（抽屉原理）
- 能否在 `O(n)` 时间内求解？（Floyd 龟兔赛跑）

## 2-1：数组中的第 K 大元素

给定整数数组 `nums` 和整数 `k`，返回数组中第 `k` 大的元素（非第 `k` 个不同元素）。你能在不整体排序的情况下完成吗？

### 示例
- 输入：`nums = [3,2,1,5,6,4], k = 2` → 输出：`5`
- 输入：`nums = [3,2,3,1,2,4,5,5,6], k = 4` → 输出：`4`

### 约束
- $1 \le k \le \text{nums.length} \le 10^5$
- $-10^4 \le \text{nums}[i] \le 10^4$

## 3-1：医院诊室分配系统

设计一个系统，将病人分配到 `k` 个诊室，并最终生成一个用于取药的全局有序队列。

### 问题背景
- 有 `k` 个诊室、`n` 个已挂号病人（编号为 `1..n`）。
- 病人按到达顺序组成到达列表（顺序与编号大小无关）。
- 需要满足：
	- 负载均衡：任意两个诊室的病人数目差不超过 `1`；
	- 诊室内有序：每个诊室内病人按编号升序维护；
	- 最终合并：将全部诊室队列合并为一个按编号升序的全局队列用于取药。

### 任务要求（实现 HospitalSystem 类）
- `assign_patients_to_clinics(arrivals: List[int], k: int) -> List[List[int]]`
	- 在遍历到达序列时进行分配，保持每个诊室内部的编号升序与全局负载均衡；
	- 当多个诊室当前负载相同时，优先分配到编号更小的诊室（稳定且可复现）。
- `merge_clinic_queues(queues: List[List[int]]) -> List[int]`
	- 使用分治法（两两归并）将 `k` 个已排序诊室队列合并为一个升序队列；
	- 目标复杂度 `O(n log k)`。
- `process_hospital_queue(arrivals: List[int], k: int) -> List[int]`
	- 主流程：分配 → 合并 → 返回最终队列。

### 约定（输入/输出）
- 输入：到达序列 `arrivals`（长度为 `n`，元素为互不相同的编号 `1..n`），整数 `k`；
- 输出：长度为 `n` 的数组，按病人编号升序排列；
- 边界：`k > n` 时允许部分诊室为空；`k = 1` 时等价于不做分配。

### 约束
- $1 \le k \le 10^3$
- $1 \le n \le 10^5$
- 病人编号唯一，范围 $1..n$
